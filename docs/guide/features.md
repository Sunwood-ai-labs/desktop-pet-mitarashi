# Features and Controls

## Mascot Modes

The tray exposes three behavior modes:

- `Running`: the mascot walks automatically around the outer edges of the current display.
- `Idle`: the mascot stays visible but stops walking.
- `Random`: the mascot alternates between running and idle on a timer.

The mascot window stays always on top and click-through, so it remains visible without blocking clicks on your other apps.

## Speed Control

You can change the speed from the tray menu:

- Use the tray items `Speed: Fast`, `Speed: Medium`, and `Speed: Slow`.

## Background Illustration

The tray item `Show Background` reveals a wide background illustration behind the mascot. The background is non-interactive so it does not block clicks on other windows.

## Tray Interaction

- Double-click the tray icon to reveal the mascot window without taking focus.
- Use `Show` when the window is hidden.
- Use `Quit` to exit fully instead of minimizing to the tray.
