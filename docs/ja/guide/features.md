# 機能と操作

## マスコットのモード

トレイには 3 つの挙動モードがあります。

- `Running`: モニターの上下左右を自動で歩き続けます。
- `Idle`: その場にとどまり、歩行を停止します。
- `Random`: 一定時間ごとに Running と Idle を切り替えます。

手動モード中は、マスコットをクリックすると `Running` と `Idle` を切り替えられます。ドラッグ操作はクリック切替と干渉しないように分離されています。

## 速度変更

速度はトレイメニューから変更できます。

- トレイの `Speed: Fast`、`Speed: Medium`、`Speed: Slow` を使う

## 背景イラスト

`Show Background` を有効にすると、マスコットの後ろに横長の背景イラストを表示できます。背景はクリックを奪わない設定なので、他のアプリ操作を邪魔しません。

## トレイ操作

- トレイアイコンのダブルクリックでマスコットを再表示
- `Show` で隠れたウィンドウを再表示
- `Quit` で完全終了
